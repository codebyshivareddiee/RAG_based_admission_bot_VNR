MCA PROGRAMME (MASTER OF COMPUTER APPLICATIONS)
================================================================================

DURATION: 2 Years Full-Time

DEPARTMENT: Information Technology (IT)

YEAR ESTABLISHED: To be confirmed

INTAKE CAPACITY: To be confirmed

PROGRAMME OVERVIEW:
The Department of Information Technology offers a 2-year postgraduate programme in Master of Computer Applications (MCA), designed to equip students with advanced knowledge and skills in computer science, software development, and information technology.

MCA focuses on building problem-solving abilities, analytical thinking, and programming expertise, preparing students to work in diverse IT roles.

KEY CURRICULUM AREAS:

CORE PROGRAMMING CONCEPTS:
- C, C++ Programming
- Java Programming
- Python Programming

SOFTWARE DEVELOPMENT:
- Software Engineering
- Development Methodologies (Agile, DevOps)
- Software Project Management

DATABASE & SYSTEMS:
- Database Management Systems (DBMS)
- Computer Networks
- Operating Systems

WEB & CLOUD TECHNOLOGIES:
- Web Technologies (HTML, CSS, JavaScript, Frameworks)
- Cloud Computing (AWS, Azure, Google Cloud)
- Web Services and APIs

ALGORITHMS & DATA STRUCTURES:
- Data Structures and Algorithms
- Design and Analysis of Algorithms
- Problem Solving Techniques

EMERGING TECHNOLOGIES:
- Artificial Intelligence
- Machine Learning
- Big Data Analytics
- Data Mining

PRACTICAL TRAINING:
- Project Work (Industry-oriented)
- Internships (Hands-on experience)
- Live Projects
- Industry Collaborations

CAREER OPPORTUNITIES:
MCA graduates are prepared for diverse IT roles including:
- Software Developer/Engineer
- Full Stack Developer
- Data Analyst
- Database Administrator
- System Analyst
- Cloud Engineer
- ML Engineer
- IT Consultant
- Mobile App Developer
- Web Developer
