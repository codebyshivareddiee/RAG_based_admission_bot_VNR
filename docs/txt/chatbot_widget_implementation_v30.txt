## Technical Implementation:

### Speech Recognition (STT) Functions:
```javascript
function toggleSpeechRecognition() - Main toggle function
function startContinuousListening() - Starts continuous listening mode
function stopListening() - Stops speech recognition
function updateListeningUI(listening) - Updates UI feedback
function startAutoStopTimer() - Manages auto-stop functionality
function resetAutoStopTimer() - Resets silence timer on speech
```

### Text-to-Speech (TTS) Functions:
```javascript
function speakText(text, buttonElement) - Main TTS function
function stopSpeech() - Stops current speech
function resetSpeechState(buttonElement) - Resets TTS state
function detectLanguageFromText(text) - Language detection (backup)
```

### Auto-Stop Logic:
- Monitors speech activity with lastSpeechTime tracking
- 4-second silence detection with 1-second check intervals
- Timer resets on any speech activity (interim or final)
- Prevents infinite listening and battery drain

### Voice Matching Algorithm:
1. Exact language match (e.g., "te-IN")
2. Language prefix match (e.g., "te")
3. Voice name matching (e.g., "telugu", "తెలుగు")
4. Fallback to default system voice
