### Key Animations:

#### Listening Glow Animation:
```css
@keyframes listeningGlow {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}
```
- Creates colorful flowing glow around input area during speech recognition
- Uses linear gradient with multiple colors
- 2-second duration with smooth easing

#### Input Pulse Animation:
```css
@keyframes inputPulse {
  0%, 100% { box-shadow: 0 0 0 2px rgba(244, 67, 54, 0.2); }
  50% { box-shadow: 0 0 0 4px rgba(244, 67, 54, 0.4); }
}
```
- Pulsing red border effect during listening
- Subtle shadow expansion and contraction
- Synchronized with listening state

#### Status Fade-In:
```css
@keyframes statusFadeIn {
  from { opacity: 0; transform: translateY(5px); }
  to { opacity: 1; transform: translateY(0); }
}
```
- Smooth appearance of status text
- Upward slide motion with opacity change
- 0.3-second duration for snappy feedback
